# Workflow: UI 구현

## 목표
학습 트래커의 전체 UI를 React 컴포넌트로 구현한다.

## 입력값
- 커리큘럼 JSON 데이터
- Firestore 진도 데이터

## 단계

### Step 1: 레이아웃 컴포넌트
- Header (앱 제목, 네비게이션)
- 반응형 컨테이너 (모바일/PC)

### Step 2: 대시보드 페이지
- 전체 진도율 프로그레스 바
- 트랙별 진행 카드 (완료/전체 표시)
- 상태별 요약 (미학습, 완료, 나중에)

### Step 3: Day 목록 뷰
- Phase별 Day 카드 리스트
- 상태 아이콘 표시 (미학습/완료/나중에)
- 상태별 필터링

### Step 4: Day 상세 뷰
- 학습 내용 + 생성 예시 표시
- 상태 토글 버튼 (3종)
- 메모 입력/저장 영역

## 결과물
- 모든 페이지가 React Router로 연결
- 반응형 UI (모바일/PC)

## 에러 대응
- 라우팅 에러 → React Router 설정 확인
- 스타일 깨짐 → Tailwind 클래스 확인
